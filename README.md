#### 项目说明
市面上丰富的图表绘制工具足以让有明确设计（“我需要一张柱状图，其中x轴为天气，y轴为平均降水量”）的用户方便地完成图表制作，而本项目针对的是（“我想看看天气和降水量的关联”）的情况。

在此基础上，Draco的推荐算法会根据用户的需求和数据生成合适的可视化图形供用户筛选.需要注意的是,本项目只为用户提供多样化的可视化样式推荐,而不是满足用户具体的绘制需求,若有明确的绘图需求还请使用其他图表绘制工具.
